The following information is for running the framework

required softwares
1. python
2.pytest
3.pytest-html
4. any ide(example pycharm)

open your pycharm and load the project'Framework'

configure your settings for default test runner as pytest

add the following in run configurations in pychram for additional attributes '--html=Reports/Report.html  --self-contained-html' 
and make it as project configuuration file


run the testcases.py file

analyze the results in run as well as using the html reports available under 'test\reports\report.html' file
